## 🎛️ **CMS 관리 시스템**

### 📝 **콘텐츠 관리**
- **홈페이지**: 히어로, 대표 인사말, 정책 소개
- **당 소개**: 창당 취지, 강령, 조직도, 인물 소개
- **정책**: 경제, 교육, 안보 정책 상세 관리
- **소식**: 공지사항, 보도자료, 행사 정보
- **참여**: 당원 가입, 당비, 후원 안내
- **자료실**: 다운로드, 정책자료, 홍보물

### 🔧 **관리자 도구**
- **CMS 관리자**: `/freedom-control` 접속
- **로그 뷰어**: `/admin/log-viewer.html`
- **아카이브 관리**: `/admin/archive-viewer.html`
- **실시간 모니터링**: 모든 활동 추적
